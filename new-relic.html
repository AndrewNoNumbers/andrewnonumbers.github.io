What is New Relic?

New Relic (NEWR) started with application performance monitoring but now sells a full set of tools that help development teams monitor every aspect of their infrastructure. Developers install New Relic "agents" in their code, and then they can use New Relic's web software to troubleshoot, monitor, and debug performance issues.
